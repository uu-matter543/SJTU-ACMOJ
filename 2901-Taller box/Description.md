# 2901 最近的更高箱子

## 2901-题目描述

仓库里有一排箱子，第 `i` 个箱子的高度是 `h_i`。求：对于每一个箱子 `i`，它**右边第一个比它高的箱子**的编号。如果右边没有比它高的箱子，则输出 `0`。

温馨提示：需要设计复杂度为 $O(n)$ 的算法实现。

## 2901-输入格式

- 第一行：输入一个整数 `n` ，表示箱子的数量。
- 第二行：`n` 个整数 `h_1, h_2, ..., h_n`，表示每个箱子的高度。
- 数据范围
  - $1 \leq n \leq 10^5$
  - $1 \leq h_i \leq 10^9$

## 2901-输出格式

- 输出 `n` 个整数，第 `i` 个数表示第 `i` 个箱子右边第一个比它高的箱子的编号。
- 如果不存在，则输出 `0`。
